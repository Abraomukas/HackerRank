# Java Loops I

Given an integer, **N**, print its first **10** multiples. Each multiple **N x i** (where **1 <= i <= 10**) should be printed on a new line in the form: _N x i = result_.

### Input format

A simple integer, **N**.

### Constrains

- **2 <= N <=20**

### Sample input

```
2
```

### Sample output

```
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
2 x 4 = 8
2 x 5 = 10
2 x 6 = 12
2 x 7 = 14
2 x 8 = 16
2 x 9 = 18
2 x 10 = 20
```
